# Career Agent AI 🤖

Advanced AI-powered career management & salary negotiation system. Built with Claude, LangChain, and Next.js.

## Quick Start

```bash
npm install
cp .env.example .env
npm run db:migrate
npm run dev
```

## Modules

- 💼 **Salary Negotiation Engine** - AI-driven offer analysis
- 📊 **Market Intelligence** - Real-time salary data
- 🎥 **Mock Interviews** - AI interview practice
- 📈 **Career Optimizer** - 3-5 year planning
- 🔗 **Network Tools** - Referral mapping
- 📱 **Dashboard** - Track offers & progress

See [docs/NEGOTIATION_ENGINE.md](docs/NEGOTIATION_ENGINE.md) for details.
